Dofus Treasure Hunt Clues

I created a script to extract these images and clues from a thread on ImpsVillage started by UltimateSensation.

The script assumes that clues are at most 9 words long, start with letters, contain only letters, dash, comma, apostrophe, parentheses, and whitespace, and always appear before their images. If you see an image you posted that has the wrong clue here, you could edit your post following these rules, and next time I update this page it will hopefully get corrected.

You can use the box on the top-left corner to filter the clues. You can also add this as a search engine in your browser.
